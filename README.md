# Derma Assist Front-End

The Derma Assist Front-End is the mobile application interface for a dermatology assistant project. It is built using modern Android standards, focusing on a clean, intuitive, and user-friendly experience.

- **Modern UI/UX**: Designed for smooth and easy navigation.

- **Front-end Ready**: Prepared for future integration with a robust backend or AI/Machine Learning services.
